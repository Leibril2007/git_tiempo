Comandos de git

Número Comando Descripción
1. git init Inicializa un repositorio Git en el directorio actual
2. git add Agrega un archivo al área de preparación (staging)
3. git commit -m "mensaje" Guarda los cambios en el repositorio con un mensaje descriptivo
4. git pull Obtiene cambios del repositorio remoto y los fusiona con el repositorio local
5. git push Envía los cambios locales al repositorio remoto
